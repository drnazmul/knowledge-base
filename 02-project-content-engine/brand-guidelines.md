- Hex: `#FAFAFA`

- Usage: Primary background, clean canvas

- Hex: `#F0F4F8`
- RGB: 240, 244, 248
- Usage: Alternate background, sections, subtle divisions

- Hex: `#334155`

- Hex: `#64748B`

- Hex: `#FFFFFF`

**Do's:**
‚úÖ Use Bright Blue for main headings and primary actions  
‚úÖ Use Teal for scientific/technical elements and data visualization  
‚úÖ Use Coral sparingly for important CTAs and emphasis  
‚úÖ Maintain high contrast ratios for accessibility (4.5:1 minimum)  
‚úÖ Use neutrals for text and backgrounds  

**Don'ts:**
‚ùå Don't use colors at full saturation on large areas

## üî§ TYPOGRAPHY

### **Digital/Social Media (LinkedIn, Blog, Website)**

**Primary Font Family:**
- **Inter** (preferred) or **Poppins** (alternative)
- Modern, clean, highly readable
- Excellent for screens
- Professional yet approachable

**Usage:**
- **Headlines:** Inter Bold, 48-72px
- **Subheadings:** Inter SemiBold, 24-36px
- **Body Text:** Inter Regular, 16-18px
- **Captions:** Inter Regular, 14px
- **Buttons/CTAs:** Inter Medium, 16px

**Line Height:**
- Headlines: 1.2
- Body text: 1.6-1.8
- Captions: 1.5

**Letter Spacing:**
- Headlines: -0.02em (slightly tighter)
- Body: Normal
- ALL CAPS elements: +0.05em (slightly wider)

---

### **Documents & Formal Content**

**Primary Font:**
- **Calibri** or **Arial**
- Universally available
- Professional and clean
- Microsoft Office compatible

**Usage:**
- **Headings:** Calibri Bold, 18-24pt
- **Subheadings:** Calibri Bold, 14-16pt
- **Body Text:** Calibri Regular, 11-12pt
- **Footnotes:** Calibri Regular, 9pt

---

### **Academic Papers**

**Primary Font:**
- **Times New Roman** (journal standard)
- Traditional, formal
- High compatibility

**Usage:**
- Follow journal-specific guidelines
- Typically 12pt for body
- 10pt for captions/footnotes

---

### **Monospace (Code/Data)**

**Font:**
- **Fira Code** (with ligatures) or **Courier New**
- For code snippets, data tables, technical content

**Usage:**
- Code blocks: 14-16px
- Inline code: Same as body text
- Data tables: 12-14px

---

### **Typography Best Practices**

**Do's:**
‚úÖ Use maximum 2 font families per document  
‚úÖ Maintain clear hierarchy (size, weight, color)  
‚úÖ Use adequate line spacing (1.6+ for body text)  
‚úÖ Limit line length (60-80 characters for readability)  
‚úÖ Use proper punctuation (em dashes, smart quotes)  

**Don'ts:**
‚ùå Don't use more than 3 font sizes in one section  
‚ùå Don't use ALL CAPS for long text (headlines only)  
‚ùå Don't use decorative fonts  
‚ùå Don't set body text below 14px for digital  
‚ùå Don't use justified alignment for short lines

## üé® VISUAL STYLE GUIDE

### **Design Principles**

**1. Scientific Minimalism**
- Clean, uncluttered layouts
- Generous white space
- Focus on content over decoration
- Data-driven visuals

**2. Modern Professionalism**
- Contemporary design aesthetics
- Professional but not corporate
- Academic but not stodgy
- Innovative but not trendy

**3. Accessibility First**
- High contrast ratios
- Readable typography
- Clear visual hierarchy
- Mobile-friendly layouts

---

### **Layout Principles**

**Grid System:**
- Use 12-column grid for flexibility
- Consistent margins and gutters
- Align elements to grid
- Maintain visual rhythm

**White Space:**
- Generous padding (minimum 24px)
- Section spacing (48-64px)
- Element spacing (16-24px)
- Don't fear empty space

**Visual Hierarchy:**
- Clear distinction between heading levels
- Important elements larger/bolder
- Secondary information smaller/lighter
- Consistent throughout

---

### **Imagery Style**

**Photography:**
- High quality, well-lit
- Lab equipment and research environment
- Authentic (not stock photos if possible)
- Professional but not posed
- Color palette aligned with brand

**Graphics & Icons:**
- Line style (not filled)
- Consistent stroke weight (2-3px)
- Simple, clear, purposeful
- Brand colors or monochrome
- Scale consistently

**Data Visualization:**
- Clean, minimal chart styles
- Brand color palette
- Clear labels and legends
- High contrast for readability
- Export in high resolution

**Diagrams:**
- Vector graphics (scalable)
- Brand colors for highlighting
- Clear labeling
- Professional appearance
- Educational focus

---

### **Content Design Patterns**

**LinkedIn Carousels:**
- Size: 1080x1080px (square) or 1080x1350px (portrait)
- Format: PDF upload to LinkedIn
- Slides: 8-10 optimal
- Design:
  - Consistent template across all slides
  - One main point per slide
  - Generous white space
  - Brand colors for accents
  - Footer with branding on each slide
  - Clear visual hierarchy

**Blog Posts:**
- Header image: 1200x630px
- Featured image ratio: 16:9 or 3:2
- In-text images: Responsive, max-width 100%
- Consistent image treatment (borders, shadows)

**Presentations:**
- Size: 16:9 widescreen
- Template: Consistent across all presentations
- Title slide with branding
- Slide numbers and author name in footer
- Consistent heading placement
- Data visualizations in brand colors

## üìê SPACING SYSTEM

**Consistent Spacing Scale (px):**
```
4px   - Minimal (icon-text spacing)
8px   - Tight (element margins)
16px  - Base (standard padding)
24px  - Comfortable (section padding)
32px  - Loose (major element spacing)
48px  - Section spacing
64px  - Major section breaks
96px  - Hero sections
128px - Page breaks
```

**Usage:**
- Padding within elements: 16-24px
- Margins between elements: 16-32px
- Section spacing: 48-64px
- Consistent throughout all designs
