LINKEDIN CAROUSEL SPECIFICATIONS:
When creating LinkedIn carousel slides (PPTX files for upload to Canva Pro):

Size: 1080 x 1350 px (4:5 portrait ratio)
In PPTX: use custom layout with defineLayout at 7.5" x 9.375" (maintains 4:5 ratio)
Format: Portrait orientation, NOT landscape
Slides: 8-10 slides per carousel (cover + content + summary + CTA)
Structure per carousel:

Slide 1: Cover (hook title, tags, brand footer)
Slides 2-6: Content slides (one topic per slide)
Slide 7: Summary/recap slide
Slide 8: CTA slide (save, comment, follow)


Brand elements on every slide: "Chemistry&AI" footer, teal accent bar at top
Color palette: Deep navy background (0A1628), teal accent (00B4D8), mint secondary (00E5A0)
Fonts: Arial Black for headers, Calibri for body text
Always include URL/source reference in footer of tool review slides
Final output: PPTX file (user uploads to Canva Pro to convert to PDF for LinkedIn)
