## ðŸŽ¯ WORKFLOW 1: Create Cover Letter for Specific Position

**When to Use:** Applying to a faculty position

**Steps:**

1. **Gather Information:**
   ```
   I'm applying for [Position Title] at [University Name].
   
   Job posting: [paste key requirements]
   
   Deadline: [Date]
   
   Can you help me create a customized cover letter?
   ```

2. **Claude Will Ask:**
   - Which template to use? (Monash? Sunway? Public university? Private?)
   - Any specific points to emphasize?
   - Faculty you want to mention?

3. **You Provide:**
   - Template preference (or let Claude recommend)
   - Specific requirements from job posting
   - Any additional context

4. **Claude Delivers:**
   - Customized 2-3 page cover letter
   - Specific to that university
   - Ready to review and edit

5. **You Finalize:**
   - Review for accuracy
   - Add any personal touches
   - Proofread
   - Save as PDF: `Alam_CoverLetter_[University]_Jan2025.pdf`

**Time:** 30-45 minutes

## ðŸŽ¯ WORKFLOW 2: Prepare for Interview

**When to Use:** You have an interview scheduled

**Steps:**

1. **Inform Claude:**
   ```
   I have an interview at [University] on [Date].
   
   Format: [Phone screen / Campus visit]
   
   Position: [Title]
   
   Help me prepare!
   ```

2. **Claude Will Provide:**
   - University-specific talking points (from university-profiles.md)
   - Common questions with YOUR specific answers
   - Questions YOU should ask them
   - Teaching demo recommendations (if campus visit)
   - Research presentation structure (if campus visit)

3. **Practice Session:**
   ```
   Can you conduct a mock interview with me?
   Ask me 10 common questions and evaluate my answers.
   ```

4. **Claude Conducts:**
   - Mock interview
   - Asks questions
   - You respond (type your answers)
   - Claude provides feedback

5. **Refinement:**
   - Claude suggests improvements
   - You practice refined answers
   - Repeat until comfortable

**Time:** 2-3 hours (spread over several days)

## ðŸŽ¯ WORKFLOW 5: Prepare Teaching Demonstration

**When to Use:** Campus interview with teaching demo required

**Steps:**

1. **Inform Claude:**
   ```
   I need to prepare a 30-minute teaching demonstration on [topic] for [University].
   
   Audience: [Master's students / Undergraduates]
   
   Help me design it!
   ```

2. **Claude Will:**
   - Recommend topic (if flexible) - Usually NADES for you!
   - Provide complete lesson plan structure
   - Include active learning activity
   - Suggest materials needed
   - Give timing breakdown

3. **You Develop:**
   - Create slides (using Claude's outline)
   - Prepare handouts
   - Practice timing

4. **Practice with Claude:**
   ```
   I'm going to practice my teaching demo.
   Here's my outline: [paste]
   
   Provide feedback!
   ```

5. **Claude Reviews:**
   - Timing assessment
   - Content clarity
   - Engagement level
   - Improvement suggestions

**Time:** 4-6 hours (spread over 1-2 weeks)

## ðŸŽ¯ WORKFLOW 6: Evaluate Job Offer

**When to Use:** You received an offer (congratulations!)

**Steps:**

1. **Share Details:**
   ```
   I received an offer from [University]!
   
   Position: [Title]
   Salary: RM [Amount]
   Start Date: [Date]
   Teaching Load: [X courses/semester]
   Startup Package: [Details]
   
   Help me evaluate it!
   ```

2. **Claude Will:**
   - Compare to market rates (from university-profiles.md)
   - Assess teaching load (reasonable?)
   - Evaluate research support
   - Consider location/lifestyle
   - Use decision-making framework

3. **If Multiple Offers:**
   ```
   I also have an offer from [University 2].
   Here are the details: [paste]
   
   Help me compare!
   ```

4. **Claude Compares:**
   - Side-by-side analysis
   - Weighted scoring (your priorities)
   - Long-term career implications
   - Recommendation (but you decide!)

5. **Negotiation Help:**
   ```
   Should I try to negotiate?
   What's reasonable to ask for?
   ```

**Time:** 1-2 hours

## ðŸŽ¯ WORKFLOW 7: Weekly Job Search Strategy

**When to Use:** Every Monday morning

**Steps:**

1. **Review with Claude:**
   ```
   It's Monday. Let's review my job search progress.
   
   What should I focus on this week?
   ```

2. **Claude Will:**
   - List active applications and status
   - Identify upcoming deadlines
   - Suggest new positions to research
   - Recommend networking actions
   - Provide weekly task list

3. **You Plan:**
   - Which applications to submit this week (1-2)
   - Follow-ups needed
   - Networking outreach
   - Interview prep if scheduled

4. **Friday Check-In:**
   ```
   Week's progress:
   - Applied to: [X positions]
   - Interviews: [Scheduled/Completed]
   - Challenges: [Any issues]
   
   How did I do?
   ```

5. **Claude Reviews:**
   - Celebrates wins
   - Identifies improvements
   - Plans next week

**Time:** 30 minutes Monday, 15 minutes Friday

## âš¡ PRO TIPS

**Maximize Claude's Help:**

1. **Be Specific:** "Monash Malaysia Green Chemistry position" not "a job"

2. **Provide Context:** Share job posting details, deadlines, your priorities

3. **Iterate:** Don't expect perfect on first draft - refine together

6. **Leverage Templates:** Start from templates, then customize

7. **Practice:** Mock interviews are incredibly valuable

8. **Stay Organized:** Use the tracking workflows religiously
